# JavaCore.biz Website

A retro-inspired website that pays homage to the early days of the internet. Built using pure HTML for maximum nostalgia and simplicity.

## Features
- Classic HTML design
- No CSS or JavaScript
- Maximum compatibility with all browsers
- Fast loading times

## Local Development
Simply open `index.html` in any web browser to view the site.

## Deployment
The site is deployed at [javacore.biz](https://javacore.biz)
